## 🚀 Learning Highlights

### Industrial Image Processing
Mastered the use of **MATLAB** and **LabVIEW** to create robust inspection algorithms, focusing on edge detection, pattern matching, and OCR (Optical Character Recognition).

### Smart Systems & Simulation
Utilized **CoppeliaSim** to design robotic environments where vision sensors drive decision-making. Integrated **Cognex In-Sight** workflows to simulate real-world factory floor quality control.

### AI Integration
Explored the bridge between traditional computer vision and modern **Machine Learning**, applying neural networks to solve complex identification tasks that traditional rule-based algorithms cannot handle.
